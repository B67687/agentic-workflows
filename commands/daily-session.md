## Daily Session Overview

Run at session start to get oriented:

```bash
bash scripts/daily-session.sh
```

Shows:
- Active goal tree path with meso-goal progress
- Current branch, HEAD commit, dirty files, sync status
- Recent session changelog (last 3 sessions)
- Last 5 commits
- Quick command reference

Compact mode:
```bash
bash scripts/daily-session.sh --compact
```
