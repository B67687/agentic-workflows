# Archive

Historical model comparison results go here.

## Naming Convention

Format: `{model-slug}-{date}.md`

## Self-Documentation

When a new model is tested 3+ times, synthesize a comparison summary and move the source result files here.

## Comparison Format

```markdown
# Comparison: {Model A} vs {Model B}

**Date range:** YYYY-MM-DD to YYYY-MM-DD
**Tasks:** N standardized tasks

## Results Table

| Task | Model A | Model B |
|---|---|---|
| task-id-001 | Pass | Fail |
| task-id-002 | Pass | Pass |
...

## Analysis

[When each model wins, why]

## Recommendation

[Which to use for which category]
```
