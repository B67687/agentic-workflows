---
description: Turn a big goal into one bounded milestone bet
---

Use this after the north-star is clear or when a big goal needs one meaningful next bet.

Pass the plain goal on the same line, like:
`/shape-milestone recreate Elemental Battlegrounds faithfully enough to preserve the nostalgia and play feel`

First run:
`bash ./milestone-shape.sh "$ARGUMENTS"`

Then return a compact milestone note with:
- one milestone bet
- the appetite for that bet
- what proves the milestone was worth doing
- what is explicitly not in this milestone
- the first slice target
- the next command to use
